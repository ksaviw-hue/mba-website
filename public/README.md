# Public Assets

## Logo Setup

Please add your MBA logo to this directory as `logo.png` for it to appear on the website.

The logo will be displayed:
- In the navigation bar (40x40px)
- As the browser favicon/tab icon

Recommended specifications:
- Format: PNG with transparent background
- Size: At least 200x200px (will be scaled down)
- Aspect ratio: Square (1:1)

Once you add the logo file as `logo.png`, it will automatically appear throughout the site.
