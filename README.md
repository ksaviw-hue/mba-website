# EBA Basketball League Website

A modern basketball league website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Home Page**: Latest news, articles, and upcoming games
- **Branding**: Team information and ownership
- **Stats**: League leaders and statistics
- **Games**: Schedule and results
- **Player Profiles**: Detailed player statistics and info
- **Admin Panel**: Manage players, teams, and content

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This website is ready to deploy on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy!

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React
